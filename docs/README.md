# 기능 요구 사항

- [] 게임 시작 문구를 출력하는 기능
- []상대방 값을 생성하는 기능
    - [] 1 ~ 9까지 중복되지 않는 랜덤한 값
- [] 사용자의 정답값에대한 입력을 받는 기능
- [] 입력값이 올바른지 체크하는 기능
    - [] 3자리인지
    - [] 숫자인지
    - [] 중복된 수를 입력하였는지
- [] 사용자의 정답 값과 상대방 값을 비교하는 기능
    - [] 답이 다르면 힌트 메세지를 제공
    - [] 답이 같으면 축하 메세지를 출력
- [] 게임 재도전 여부를 물어보는 기능
- [] 재도전 하면 게임이 다시 시작하는 기능
- [] 재도전 거부시 게임이 종료하는 기능
